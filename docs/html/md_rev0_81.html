Raspberry Pi Pico Eğitim Kartı

Genel Bakış

Bu eğitim kartı, Raspberry Pi Pico ile C programlama ve Pico-SDK eğitimi için tasarlanmış 12 farklı komponent grubunu içerir. Her komponent grubu için gerekli pin tanımlamaları ve kullanım örnekleri aşağıda belirtilmiştir.

Pin Grupları ve Fonksiyonları

1. Analog Girişler

  • Keypad (GPIO28)
    • 3x4 matris keypad için analog okuma

    • Tuş takımı düzeni:

      | 1 | 2 | 3 |
      | 4 | 5 | 6 |
      | 7 | 8 | 9 |
      | * | 0 | # |
    • keypadOku() fonksiyonu ile kontrol edilir
    • ADC değer aralıkları tanımlıdır
  • Potansiyometre (GPIO27)
    • Analog değer okuma
    • 0-4095 arası ADC değeri
    • read_analog() fonksiyonu ile okunur
  • LDR Sensör (GPIO26)
    • Işık seviyesi ölçümü
    • Analog değer okuma
    • read_analog() fonksiyonu ile okunur

2. Dijital Butonlar

  • Buton Grubu
    • BUTTON_UP (GPIO5): Yukarı ok tuşu
    • BUTTON_OK (GPIO6): Onay/Enter tuşu
    • BUTTON_DOWN (GPIO7): Aşağı ok tuşu
    • Debounce korumalı okuma
    • button_pressed() ve check_button_event() fonksiyonları ile kontrol

3. LED Çıkışları

  • Basit LED'ler
    • RED (GPIO4): Kırmızı LED
    • YELLOW (GPIO3): Sarı LED
    • GREEN (GPIO2): Yeşil LED
    • Dijital çıkış veya PWM kontrolü
  • RGB LED (PWM Kontrollü)
    • RGB_R (GPIO16): Kırmızı kanal
    • RGB_G (GPIO17): Yeşil kanal
    • RGB_B (GPIO18): Mavi kanal
    • PWM ile parlaklık kontrolü
    • set_led_pwm() fonksiyonu ile kontrol

4. Ses Çıkışı

  • Buzzer (GPIO10)
    • PWM kontrollü ses üretimi
    • Nota çalma kapasitesi
    • play_note() ve play_notes() fonksiyonları
    • Frekans ve süre parametreleri ile kontrol

5. Step Motor

  • 4-Pin Kontrol
    • A1 (GPIO11)
    • A2 (GPIO12)
    • B1 (GPIO13)
    • B2 (GPIO14)
    • Tam adım ve yarım adım kontrol
    • stepper_move() fonksiyonu ile kontrol

6. Sensör Modülleri

  • Ultrasonik Sensör
    • TR (GPIO8): Trigger pin
    • EC (GPIO9): Echo pin
    • Mesafe ölçümü yapılır
    • measure_distance() fonksiyonu ile kontrol
  • PIR Sensör (GPIO19)
    • Hareket algılama
    • Dijital giriş
    • check_motion() fonksiyonu ile kontrol

7. Haberleşme Modülleri

  • Bluetooth
    • TX (GPIO0)
    • RX (GPIO1)
    • UART haberleşme
  • I2C LCD Display
    • SDA (GPIO20)
    • SCL (GPIO21)
    • Adres: 0x27
    • 2x16 karakter gösterim
    • lcd_init(), lcd_clear(), lcd_string() fonksiyonları

Başlatma ve Konfigürasyon

Tüm komponentler init_board() fonksiyonu ile başlatılır. Bu fonksiyon:

  • ADC başlatma
  • PWM konfigürasyonu
  • GPIO yönlendirmeleri
  • I2C başlatma
  • LCD başlatma işlemlerini gerçekleştirir.

4. Kullanım Notları

Başlatma

  • LED'leri kullanmadan önce init_board() fonksiyonunu çağırın
  • PWM kontrolü için ilgili pinlerin PWM modunda olduğundan emin olun

Güç Tüketimi

  • Tüm LED'leri aynı anda maksimum parlaklıkta kullanmaktan kaçının
  • Uzun süreli kullanımlarda düşük parlaklık tercih edin

PWM Değerleri

  • PWM değer aralığı: 0-4095
  • 0: Tamamen kapalı
  • 4095: Tam parlaklık
  • 2048: %50 parlaklık

Bu fonksiyonlar ile LED'leri tek tek kontrol edebilir, renk geçişleri yapabilir ve çeşitli ışık efektleri oluşturabilirsiniz. RGB LED için renk karışımlarını deneyerek farklı renkler elde edebilirsiniz.

Bu kılavuz, LED kontrolü için temel kullanımdan ileri seviye uygulamalara kadar geniş bir yelpazede örnek ve açıklama sunuyor. Kullanıcılar bu örnekleri temel alarak kendi LED kontrol uygulamalarını geliştirebilirler.

Güvenli Kullanım

  1. Başlatma Kontrolü:
    • Her kullanım öncesi init_board() fonksiyonunun çağrıldığından emin olun
    • Motor pinlerinin doğru yapılandırıldığını kontrol edin

  2. Hız Limitleri:

    // Güvenli hız aralığı kullanımı
    uint safe_speed = (speed > 100) ? 100 : (speed < 1) ? 1 : speed;

  3. Acil Durum Yönetimi:

    gpio_set_irq_enabled_with_callback(BUTTON_OK,

  1. Yavaş Başlangıç/Bitiş:

    • Ani hızlanma/yavaşlama için rampa kullanın

    for(int i = 10; i <= target_speed; i += 5) {

float get_average_distance(int num_samples, int delay_ms) {
float total = 0;
int valid_samples = 0;
for(int i = 0; i < num_samples; i++) {
float distance = measure_distance();
if(distance > 0) {
total += distance;
valid_samples++;
}
sleep_ms(delay_ms);
}
return valid_samples > 0 ? total / valid_samples : -1;
}

bool detect_motion(void) {
static absolute_time_t last_trigger = {0};
const uint32_t COOLDOWN_MS = 2000; // Yeniden tetikleme için bekleme süresi
if(gpio_get(PIR_DETECTOR)) {
absolute_time_t now = get_absolute_time();
if(absolute_time_diff_us(last_trigger, now) > COOLDOWN_MS * 1000) {
last_trigger = now;
return true;
}
}
return false;
}

4. Önemli Notlar ve İpuçları

4.1. Zamanlama

  • İki yazma işlemi arasında kısa bir gecikme (örn: 5ms) kullanın
  • Ekranı temizledikten sonra 2ms bekleyin
lcd_clear();
sleep_ms(2); // Ekran temizleme için bekleme

4.2. Satır Uzunluğu

  • Her satır maksimum 16 karakter alabilir
  • Daha uzun metinler kesilebilir
// Uzun metin kontrolü
void safe_lcd_string(const char* text) {
char buffer[17]; // 16 karakter + null terminator
strncpy(buffer, text, 16);
buffer[16] = '\0';
lcd_string(buffer);
}

4.3. Özel Karakterler

  • Türkçe karakterler LCD'de farklı görünebilir
  • ASCII karakterlerini tercih edin

4.4. Güç Tüketimi

  • LCD arka ışığı sürekli açık kalır
  • Enerji tasarrufu için kullanılmadığında ekranı temizleyin
