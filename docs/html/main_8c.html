◆ core1_main()

void core1_main ( )

Core 1'de çalışacak işlev.

Bu fonksiyon RP2040'ın Core 1'inde çalışır ve multicore FIFO üzerinden Core 0'dan gelen adım motoru kontrol komutlarını işler. Üç parametre işler: yön, hız ve devir sayısı.

Fonksiyon:

  1. Core 0'dan gelen verileri bekler
  2. FIFO'dan motor parametrelerini okur
  3. Motor hareketini yürütür
  4. Core 0'a tamamlanma sinyalini geri gönderir
Not
Bu fonksiyon sonsuz bir döngüde çalışır ve Core 0'ın verileri doğru sırada göndermesini gerektirir: yön, hız, devir sayısı

◆ send_motor_parameters()

void send_motor_parameters ( motor_direction_t direction,
uint speed,
float revolutions )

Step motor parametrelerini Core1'e gönderir.

Bu fonksiyon, step motor kontrolü için gerekli parametreleri Core0'dan Core1'e multicore FIFO üzerinden gönderir. Gönderilen parametreler:

  1. Motor dönüş yönü
  2. Motor hızı
  3. Devir sayısı
Parametreler
directionMotor dönüş yönü (CW: saat yönü, CCW: saat yönü tersi)
speedMotor hızı (adım/saniye)
revolutionsYapılacak devir sayısı
Not
Fonksiyon, her parametre için FIFO'ya bloklu gönderim yapar ve Core1'in verileri almasını bekler
Ayrıca Bakınız
core1_main() Core1'de çalışan ve bu parametreleri alan fonksiyon
